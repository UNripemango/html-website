Sample Workout Program

  • Push 1

    1. Bench Press - 4 x 4-6 (leave 1-2 in the tank)
    2. Hi-to-Low Crossover - 3 x 10-12 (post-pressing adduction)
    3. DB Shoulder Press - 4 x 8-10 (standing movement)
    4. 1.5 Side Lateral Raises - 3 x 12-15 (direct side delt activation)
    5. Lying Triceps Extensions 3 x 10-12 into DB Waiter Curls - 3 x 10-12
    6. Rotator Cuff ER - 3 x 15-20 (rotator cuff strengthening and shoulder stability / health)

  • Push 2

    1. Barbell Overhead Press - 4 x 4-6 (leave 1-2 in the tank)
    2. Underhand DB Bench Press - 3 x 8-10
    3. Abduction Rows - 3 x 10-12
    4. Floor Flys - 3 x 10-12
    5. Triceps Close Grip Bench Press - 3 x 6-8 into DB Curl of Choice - 3 x 10-12
    6. Pushup Plus - 3 x F

  • Pull 1

    1. Deadlifts - 1 x 5
    2. Chest Supported Rows - 3 x 8-10 (takes the low back out of the equation)
    3. High Pulls - 3 x 10-12 (targets the upper back muscles with healthy shoulder mechanics)
    4. Lat Pulldowns - 2-3 x 10-12 (direct lat work)
    5. BW Chin Curls - 3 x F into Overhead Triceps Extension - 3 x 10-12
    6. Angels and Devils - 3 x 15-20 (posterior chain corrective)

  • Pull 2

    1. Weighted Pullups - 3 x 6-8
    2. Snatch Grip Deadlifts - 3 x 5 (Leave 2-3 in the tank)
    3. Alt. DB Gorilla Rows - 3 x 10-12 each arm
    4. Straight Arm Pushdowns - 2-3 x 12-15
    5. Barbell Curls - 3 x 6-8 into Triceps Pushdowns - 3 x 10-12
    6. Face Pulls - 3 x 15-20

  • Legs

    1. Squats - 4 x 4-6 (Leave 1-2 in the tank)
    2. Barbell Hip Thrust - 3 x 8-10
    3. DB/BB Alt. Reverse Lunges - 2-3 x 10-12 each leg
    4. Single Leg RDL - 2-3 x 10-12 each leg rotate with Slick Floor Bridge Curls - 2-3 x F
    5. Standing DB Calf Raises rotate with Seated DB Calf Raises - 3 x 15-20 each
